Calculator

Overview

This Capital Gains Tax (CGT) Calculator helps you accurately calculate your minimum tax obligations from investing in assets like stocks or crypto. The calculator uses a linear optimisation solver to calculate the results. Simply upload your transaction data and receive a comprehensive breakdown of your capital gains, losses, and tax discounts for each financial year.

Instructions

Follow these simple steps to calculate your capital gains tax optimally. Each section below provides detailed guidance on preparing your data and understanding the results.

How to Use This Calculator

▼
  • Make sure that you have a way of downloading your history of executed trades (further explained in the next section)
  • Simply import either a CSV or Excel file of your trade history and then the tool will attempt to calculate your capital gains tax optimally
  • You will be taken to a payment portal for a $20 payment to then download an Excel file, this file will contain all the information about your calculated minimal capital gains tax obligation, possibly saving you hundreds!

Input Trade History File

▼

If you trade with NABTrade or CommSec then the process is highly automated, but as long as you can download all your trade history then any service will work, just follow the steps below

Getting Trade History from NABTrade

▼
  • Access your account on the NABTrade website
  • On the home page, select the "Statements and Reports" button on the right hand side
  • Select the "Transaction report" button, this will take you to a page to select a date range
  • Ensure that the date range you input includes all trades you have ever done on NABTrade
  • Select the "Next" button and then the "Export Excel" button to download your report
  • Important: This is the Excel file you will upload but make sure to save it as a .xlsx file!
  • Now go to the bottom of this page and import that Excel file to calculate your capital gains tax!

Getting Trade History from CommSec

▼
  • Access your account on the CommSec website
  • Navigate to the "Accounts" page and then click on the "Transactions" tab
  • Now select a date range, ensure that the date range you input includes all trades you have ever done for that account
  • Now press the download button and make sure to download as a CSV, below is an example of what it should look like
  • If you want to combine trades from multiple accounts then follow the above steps for the other accounts and combine the trades into a single CSV
  • Now go to the bottom of this page and import that CSV file to calculate your capital gains tax!

Getting Trade History from Anywhere Else

▼
  • Download your trade history from whatever service you use, ideally in a CSV or Excel format
  • Create an Excel file or modify the file you have downloaded to match the format shown below
  • Make sure that the column names and value formats are exactly the same as the example below
  • Remember that the value for transaction_amount should include any trading fees
  • Save the file as a .csv or .xlsx file and then import it using the button below to calculate your capital gains tax!

Problems with Input

▼
  • You may get a dialog saying that short selling is detected, this may not be a problem as you may have been awarded stock without explicitly buying it, however there is a chance that the calculator missed a symbol change or a stock split, so please review the stock in question to ensure that short selling is applicable
  • In the event that a symbol change has not been accounted for by the calculator, then search online for "ticker change" in reference to that symbol, and update all instances of the symbol in your trade history file with the correct new symbol
  • Avoid including any empty rows or null values if you are manually putting together the input or adding data from different accounts into one CSV or Excel sheet
  • Make sure that the columns of the input CSV or Excel sheet format are exactly as described in the above section for the relevant input type (NABTrade, CommSec, or other)

Interpreting Results

▼
  • After uploading the correct input file and completing payment, an Excel file will be downloaded by your browser which will contain the results of the calculator
  • The file will have multiple sheets, one for each financial year that you have executed trades
  • At the bottom of each sheet there is a breakdown of your total capital gain, loss, and capital gains discount for the year. These are the exact fields you need to fill out in your tax form for capital gains from assets such as stocks or crypto
  • If you find that in previous years you have paid too much capital gains tax, you can add the overpaid amount to the "Unapplied Net Capital Losses Carried Forward" section in your tax form, and if your loss outweighs your gain, you should also include it in that section
  • In the "Calculator Result" section, click on the dropdown and you will see an image of an example Excel file that this calculator generates
