Follow these simple steps to calculate your capital gains tax. Each section below provides detailed guidance on preparing your data and understanding the results.

Input Trade History File

▼

If you trade with NabTrade or CommSec then the process is highly automated, but as long as you can download all your trade history then any service will work, just follow the steps below

Getting Trade History from NabTrade

▼
  • Access your account on the NabTrade website
  • On the home page, select the "Statements and Reports" button on the right hand side
  • Select the "Transaction report" button, this will take you to a page to select a date range
  • Ensure that the date range you input includes all trades you have ever done on NabTrade
  • Select the "Next" button and then the "Export Excel" button to download your report
  • Important: This is the Excel file you will upload but make sure to save it as a .xlsx file!
  • Now go to the bottom of this page and import that Excel file to calculate your capital gains tax!

Getting Trade History from CommSec

▼
  • Access your account on the CommSec website
  • Navigate to the "Accounts" page and then click on the "Transactions" tab
  • Now select a date range, ensure that the date range you input includes all trades you have ever done for that account
  • Now press the download button and make sure to download as a CSV, below is an example of what it should look like
  • If you want to combine trades from multiple accounts then follow the above steps for the other accounts and combine the trades into a single csv
  • Now go to the bottom of this page and import that CSV file to calculate your capital gains tax!

Getting Trade History from Anywhere Else

▼
  • Download your trade history from whatever service you use, ideally in a CSV or Excel format
  • Create an Excel file where you will save all of your executed trades
  • Make sure that the column names and value format is exactly the same as the example below
  • Remember that the value for transaction_amount should include any trading fees!
  • Save the file as a .xlsx file and then import it using the button below to calculate your capital gains tax!

Problems with Input

▼
  • You may get a dialog that short selling is detected, this can be no problem as you may have been awarded stock without explicitly buying it, however there is a chance that the calculator missed a stock split, or ticker change, or the stock was delisted, so please review the stock in question to ensure that short selling is applicable
  • Avoid including any empty rows or null values if you are manually putting together the input or adding data from different accounts into one CSV or Excel sheet
  • Make sure that the columns of the input CSV or Excel sheet are exactly as described in the above section for the relevant input type (NabTrade, CommSec, or other)

Interpreting Results

▼
  • Below is an image of an example Excel file that you will get after providing the correct input and payment
  • The file will have mutiple sheets, one for each financial year that you have executed trades
  • At the bottom of each sheet there is a breakdown of the your total capital gain, loss, and capital gains discount for the year. These are the exact fields you need to fill out in your tax form for capital gains from non-physical assets such as stocks or crypto
  • If you find that in previous years you have paid too much capital gains tax, you can add the overpaid amount to the "Unapplied Net Capital Losses Caried Forward" section in your tax form, and if your loss outweighs your gain you would also put it in that section
